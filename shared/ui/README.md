# Shared UI (Pure HTML/CSS/JS)

Run from project root:

```bash
python -m http.server 8080
```

Open in browser:

- Section 0: `http://localhost:8080/shared/ui/index.html`
- Section 1: `http://localhost:8080/shared/ui/index.html?max_section=1`
- Section 3: `http://localhost:8080/shared/ui/index.html?max_section=3`

`max_section` controls unlocked features.
